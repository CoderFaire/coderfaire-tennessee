We’re Back

And we’ve missed you.

We took 2015 off, but we’re back and ready to have another great conference with the Tennessee developer community.

Call for Papers

Are you a Tennessee-based developer? We want to hear you talk about your passion for technology. CoderFaire is a language-agnostic conference, so no matter what stack, discipline, or hobby you are working on, we want to hear from you.

LOCATION

Nashville Software School

To celebrate CoderFaire’s return, we wanted to find a venue that allows us to host the same great conference but in a more intimate setting. So, this year we decided to partner with Nashville Software School to host the conference.

Still 4 Tracks

There’s so much technology and so little time, and each year the talk submissions only get better. To fit in as much awesome as possible, we’ll be running four tracks on Saturday, as in previous years.

Great Keynotes

We’ve got some special people we’ve invited to help kick things off and close the day out. We’ll be announcing them in the future.

The Hackathon is Back

We’ve missed it and we’ve heard tell you’ve missed it, so it’s coming back. This year we’ll be bringing back the Sunday hackathon with Lab Partners and all.

After Party

Oh yeah! The famous after party is back and will be a great time. We’re not sure which zoo animals we’ll be hanging with this year, but no matter which ones, it’s sure to be a great time.

FRIENDLY PEOPLE ONLY

Code of Conduct

CoderFaire is a conference for all developers, from all backgrounds. Except jerks.

To ensure everyone has a good time, we want to set our behavior expectations up front. Basically, don’t be a jerk, and everyone will have a great time.

CoderFaire is governed by the Python Code of Conduct. More information is available at https://www.python.org/psf/codeofconduct/

To report a violation, please contact us via email before the event at organizers@coderfaire.com. For the day of the conference, we’ll provide a phone number to call, or you may find any organizer wearing a red shirt.
